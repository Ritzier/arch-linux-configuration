# Stylus

1. [Firefox Stylus](https://addons.mozilla.org/en-US/firefox/addon/styl-us/)

1. [`import.json`](https://github.com/catppuccin/userstyles/releases/download/all-userstyles-export/import.json)

1. Open the `Stylus` "manage" page

1. On the sidebar panel, click the **Import** button and select the downloaded file (`import.json`)
